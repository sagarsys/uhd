# uhd
UHD Quiz App with NodeJS/Express server

### Dependencies 
- Node v8+

### Installation
- Clone repo
- `npm i`

### Scripts
- `npm start`
    - Start express server without debug mode
- `npm run dev`
    - Start express server with debug mode
- `npm run build-js`
    - Bundle client side JS (minified version)
- `npm run dev-js`
    - Bundle client side JS
